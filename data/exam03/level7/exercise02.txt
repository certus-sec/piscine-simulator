Assignment name  : stress_test
Expected files   : stress_test.c
Allowed functions: write, malloc, free

Write a program that allocates and fills a large array (size as argument), then verifies all values are correct. Display "ok" or "fail" followed by a newline.

If the number of arguments is not 1, display a newline.

Example:
$> ./stress_test 1000000
ok
$> ./stress_test | cat -e
$
