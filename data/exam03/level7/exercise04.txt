Assignment name  : memory_stress
Expected files   : memory_stress.c
Allowed functions: malloc, free, write

Write a program that allocates and frees memory in a random pattern for n iterations (argument). Detect any double-free or leak. Display "ok" or "fail" followed by a newline.

If the number of arguments is not 1, display a newline.

Example:
$> ./memory_stress 1000
ok
$> ./memory_stress | cat -e
$
