Assignment name  : tab_mult
Expected files   : tab_mult.c
Allowed functions: printf
--------------------------------------------------------------------------------

Write a program that takes a number in a string as an argument, and displays
its multiplication table, starting from 1. If the number of parameters is not
correct or if the parameter is not a valid number, just display a newline.

Example:
$> ./tab_mult 3
3 * 1 = 3
3 * 2 = 6
3 * 3 = 9
...
3 * 9 = 27
