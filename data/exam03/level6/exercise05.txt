Assignment name  : data_structures_adv
Expected files   : data_structures_adv.c
Allowed functions: malloc, free, write

Write a program that implements a binary search tree from integers passed as arguments, then prints it using in-order traversal, separated by spaces, followed by a newline.

If the number of arguments is less than 1, display a newline.

Example:
$> ./data_structures_adv 5 3 7 1 4
1 3 4 5 7
$> ./data_structures_adv | cat -e
$
