Assignment name  : file_system_sim
Expected files   : file_system_sim.c
Allowed functions: malloc, free, write

Write a program that simulates a basic file system with mkdir, touch, ls, and cd operations. Take commands as arguments and display the result or "error".

Example:
$> ./file_system_sim mkdir home
ok
$> ./file_system_sim touch home/file.txt
ok
$> ./file_system_sim ls home
file.txt
$> ./file_system_sim | cat -e
$
