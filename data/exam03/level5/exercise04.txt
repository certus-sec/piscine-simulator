Assignment name  : algorithm_mix
Expected files   : algorithm_mix.c
Allowed functions: write, malloc, free

Write a program that takes a list of integers and finds the majority element (appears more than n/2 times). If none exists, display "none". Display the result followed by a newline.

If the number of arguments is less than 1, display a newline.

Example:
$> ./algorithm_mix 3 3 4 2 3 3 3
3
$> ./algorithm_mix 1 2 3 4
none
$> ./algorithm_mix | cat -e
$
