Assignment name  : recursion_pro
Expected files   : recursion_pro.c
Allowed functions: write

Write a recursive function that solves the Tower of Hanoi problem for n disks.

Your function must be declared as follows:
void	hanoi(int n, char from, char to, char aux);

Write a main that takes a number as argument and prints all moves, one per line.

Example:
$> ./recursion_pro 2
Move disk 1 from A to B
Move disk 2 from A to C
Move disk 1 from B to C
$> ./recursion_pro | cat -e
$
