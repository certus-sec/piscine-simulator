Assignment name  : string_master
Expected files   : string_master.c
Allowed functions: write, malloc, free

Write a program that takes a string and performs the following transformations:
1. Remove all duplicate characters
2. Sort remaining characters alphabetically
3. Display the result followed by a newline

If the number of arguments is not 1, display a newline.

Example:
$> ./string_master "hello world"
 dehlorw
$> ./string_master "abcabc"
abc
$> ./string_master | cat -e
$
