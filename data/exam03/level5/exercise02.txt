Assignment name  : parsing_engine
Expected files   : parsing_engine.c
Allowed functions: write, malloc, free

Write a program that takes a string and parses it into tokens based on multiple delimiters (space, tab, comma, semicolon). Display each token on a separate line.

If the number of arguments is not 1, display a newline.

Example:
$> ./parsing_engine "hello,world;foo bar"
hello
world
foo
bar
$> ./parsing_engine | cat -e
$
