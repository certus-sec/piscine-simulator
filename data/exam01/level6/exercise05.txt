Assignment name  : recursion_intro
Expected files   : recursion_intro.c
Allowed functions: write

Write a recursive function that calculates the factorial of a positive integer.

Your function must be declared as follows:
int	factorial(int nb);

Write a main that takes a number as argument and prints the result followed by a newline.

Example:
$> ./recursion_intro 5
120
$> ./recursion_intro 0
1
$> ./recursion_intro | cat -e
$
