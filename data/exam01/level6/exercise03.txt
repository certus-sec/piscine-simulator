Assignment name  : math_operations
Expected files   : math_operations.c
Allowed functions: atoi, printf, write

Write a program that takes three strings: two numbers and an operator (+, -, *, /, %, ^ for power). Display the result followed by a newline.

If the number of parameters is not 3, display a newline.

Example:
$> ./math_operations 2 ^ 10
1024
$> ./math_operations 10 / 3
3
$> ./math_operations | cat -e
$
