Assignment name  : aff_z
Expected files   : aff_z.c
Allowed functions: write
--------------------------------------------------------------------------------

Write a program that takes a string, and displays the first 'z' character it
encounters in it, followed by a newline. If there are no 'z' characters, just
a newline. If parameters != 1, display 'z' then newline.

Example:
$> ./aff_z "zoo" | cat -e
z$
$> ./aff_z "abc" | cat -e
$
