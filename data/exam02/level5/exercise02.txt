Assignment name  : string_transform
Expected files   : string_transform.c
Allowed functions: write, malloc, free

Write a program that takes a string and transforms it by applying the following rules:
- Replace every vowel with the next vowel (a->e, e->i, i->o, o->u, u->a)
- Replace every consonant with the previous consonant
- Keep case consistent

If the number of arguments is not 1, display a newline.

Example:
$> ./string_transform "Hello World"
Gekkn Vnqkc
$> ./string_transform | cat -e
$
