Assignment name  : parsing_complex
Expected files   : parsing_complex.c
Allowed functions: write, malloc, free

Write a program that takes a string containing nested parentheses and extracts each group of balanced parentheses as a separate line.

Groups are not separated by spaces.

Example:
$> ./parsing_complex "(hello)world"
(hello)
$> ./parsing_complex "((a)b)(c)"
((a)b)
(c)
$> ./parsing_complex "(()())"
(()())
$> ./parsing_complex | cat -e
$
