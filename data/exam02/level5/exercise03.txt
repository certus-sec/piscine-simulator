Assignment name  : algorithm_core
Expected files   : algorithm_core.c
Allowed functions: write, malloc, free

Write a program that takes a list of integers and finds the longest increasing subsequence. Display the subsequence separated by spaces, followed by a newline.

If the number of arguments is less than 1, display a newline.

Example:
$> ./algorithm_core 10 9 2 5 3 7 101 18
2 3 7 101
$> ./algorithm_core 0 1 0 3 2 3
0 1 2 3
$> ./algorithm_core | cat -e
$
