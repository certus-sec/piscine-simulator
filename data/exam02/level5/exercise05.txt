Assignment name  : recursion_basic
Expected files   : recursion_basic.c
Allowed functions: write

Write a recursive function that calculates the nth Fibonacci number.

Your function must be declared as follows:
int	fibonacci(int n);

Write a main that takes a number as argument and prints the result followed by a newline.

Example:
$> ./recursion_basic 10
55
$> ./recursion_basic 0
0
$> ./recursion_basic | cat -e
$
