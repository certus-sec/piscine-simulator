Assignment name  : bit_manipulation
Expected files   : bit_manipulation.c
Allowed functions: write

Write a program that takes an integer and displays:
1. The number of set bits (1s)
2. The position of the highest set bit
3. The position of the lowest set bit

Display each on a separate line. If the number of arguments is not 1, display a newline.

Example:
$> ./bit_manipulation 42
3
5
1
$> ./bit_manipulation 1
1
0
0
$> ./bit_manipulation | cat -e
$
