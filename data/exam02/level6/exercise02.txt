Assignment name  : memory_handling
Expected files   : memory_handling.c
Allowed functions: malloc, free, write

Write a program that allocates a 2D array of integers (rows x cols), fills it with consecutive integers starting from 1, and prints it row by row.

Arguments are rows and cols. If the number of arguments is not 2, display a newline.

Example:
$> ./memory_handling 3 3
1 2 3
4 5 6
7 8 9
$> ./memory_handling | cat -e
$
