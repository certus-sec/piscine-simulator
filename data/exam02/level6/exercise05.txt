Assignment name  : file_parsing
Expected files   : file_parsing.c
Allowed functions: open, read, close, write, malloc, free

Write a program that reads a file (filename as argument) and counts the number of lines, words, and characters. Display them separated by spaces, followed by a newline.

If the number of arguments is not 1, display a newline.

Example:
$> ./file_parsing test.txt
10 20 100
$> ./file_parsing | cat -e
$
