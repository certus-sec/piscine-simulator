Assignment name  : pointer_challenge
Expected files   : pointer_challenge.c
Allowed functions: write

Write a program that takes a string and reverses it in place using only pointers (no array indexing with []).

If the number of arguments is not 1, display a newline.

Example:
$> ./pointer_challenge "hello"
olleh
$> ./pointer_challenge "abcba"
abcba
$> ./pointer_challenge | cat -e
$
