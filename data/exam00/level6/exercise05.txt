Assignment name  : bitwise_operations
Expected files   : bitwise_operations.c
Allowed functions: write

Write a program that takes an integer as argument and displays it in binary,
hexadecimal, and octal format, followed by a newline.

If the number of arguments is not 1, display a newline.

Examples:
$> ./bitwise_operations 42
Binary: 101010
Hex: 2a
Octal: 52
$> ./bitwise_operations | cat -e
$
