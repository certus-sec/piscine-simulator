Assignment name  : ulstr
Expected files   : ulstr.c
Allowed functions: write

Write a program that takes a string and reverses the case of all its letters.
Other characters remain unchanged.

You must display the result followed by a newline.

If the number of arguments is not 1, the program displays \n.

Example:
$> ./ulstr "L'eSPrit nE peUt plUs pRogResSer s'Il staGne et sI peRsIsTent VAnIte et auto-satisfaction" | cat -e
l'EspRIT Ne PEuT PLuS PrOGrESsER S'iL STAgNE ET Si PErSiStENT vaNiTE ET AUTO-SATISFACTION$
$> ./ulstr "S'enTOuRer dE sES pEnSeEs eSt uN bIenFAiT pOuR sEs pEnSeEs" | cat -e
s'ENtoUrER De Ses PeNsEes Est Un BiENfAit POUr SEs PeNsEes$
$> ./ulstr "3+35 / 5 = 10" | cat -e
3+35 / 5 = 10$
$> ./ulstr | cat -e
$
